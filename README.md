### What I learned

-To set content in the middle of screen,
-to use box-shadow.

```css
.background{
    background-color:hsl(225, 100%, 94%);
    height:648px;
    display:  flex;
    align-items: center;
    justify-content:  center;
}
button.check{
    box-shadow: 0 15px 30px 1px hsl(245, 100%, 89%);
}
```
